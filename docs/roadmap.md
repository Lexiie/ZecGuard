# Roadmap

## Hackathon MVP

- Local web app.
- Encrypted recovery package.
- 2-of-3 guardian shares.
- ZecGuard memo generation and parsing.
- Local wallet bridge.
- Bridge ACK memo scanning.
- Manual fallback.
- Readiness dashboard.
- Reconstruction drill.
- Mainnet tx proof.

## Post-Hackathon V1

- Stronger memo scanning.
- QR-based share exchange.
- Package update and revoke flows.
- Guardian replacement.
- Improved mobile UI.
- Recovery ceremony mode.

## V2

- Tauri desktop app.
- Offline mode.
- Argon2id for password-based package flows.
- Printable recovery packets.
- Wallet integration guide.

## V3

- FROST-compatible recovery ceremony research.
- Wallet SDK.
- Standardized ZecGuard memo schema.
- Guardian rotation and share refresh.
